# Roadmap

## v0.1
- App arranca localmente con Vite.
- Layout con sidebar y navegación.
- Mocks para datos de entrenamientos, finanzas, retos, ranking e historial.
- Formularios básicos para workouts y finanzas.
- Ranking y métricas de progreso.
- Páginas de ajustes y selección de usuario.

## v0.2
- SQLite local real bajo Tauri.
- Eventos de sync reales para cambios locales.
- Validaciones Zod más completas.
- Gráficos reales en el dashboard.
- Persistencia Tauri en lugar de solo mock/localStorage.

## v0.3
- Sync con GitHub privado.
- Export / backup de datos.
- Resolución de conflictos simple.
- Mejoras en la experiencia de escritorio.
